Instrucciones para validar localmente (Windows PowerShell):

1) Usar Python 3.14 y crear virtualenv llamado `.venv_fase2`:

```powershell
py -3.14 -m venv .venv_fase2
.\.venv_fase2\Scripts\Activate.ps1
python -m pip install --upgrade pip wheel setuptools
python -m pip install --only-binary=:all: -r requirements.txt
```

2) Ejecutar el validador:

```powershell
python -m myanswers.run_validation
```

Archivos relevantes en el repo:
- `myanswers/answer_0521.py` — función `limpiar_y_escalar(df, columna)`
- `myanswers/answer_0083.py` — función `entrenar_brazo_robotico(X, y)`
- `myanswers/answer_0579.py` — función `optimizar_bosque_random(X, y)`
- `myanswers/answer_0104.py` — función `pipeline_pca_logistico(X, y, n_componentes=0.95)`
- `myanswers/run_validation.py` — validador automático que compara con los pickles en `myanswers/cases/`
- `myanswers/diagnostico.txt` — registro de la validación ejecutada localmente (todas OK).

Observaciones:
- El validador se ejecutó correctamente en mi máquina usando Python 3.14 y `.venv_fase2`.
- Si no dispone de Python 3.14, puede usar `conda create -n fase2 python=3.14` y luego instalar `-r requirements.txt`.
